Setup Instructions

Get started with StarCraft: Brood War AI Development as fast as possible.

Linux / Bot Development using TorchCraft

TorchCraft comes with a example bot written in Lua or Python using BWAPI 4.2.0.

This repo comes with BWAPI, and uses bwheadless.exe and ChaosLauncher to launch StarCraft with BWAPI.

  1. CLone this repo to your computer
  2. Unxip Starcraft Broodwar 1.16.1 to the included windows/starcraft folder
  3. RUn windows/RunCBotAndStarCraft.sh which will launch the bot executable and Starcraft/BWAPI
  4. Open windows/c--/visual/StarCraft.sln in Visual Studio 2019 to modify / recompile the code.
