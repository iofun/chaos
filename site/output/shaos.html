SHA/OS

SHA/OS is a nonsense.ws sponsored project.

Our primary user base is within the distributed high-performance computing community. However, our users come from a wide variety of industries.

Our Mission: Driven by nonsense.ws we focus on multi-dimensional research, shaos.ws provide a workspace environment full of distributed asynchronous tools.

Our Goal: Provide a stable, flexible, distributed environment for kids of all ages.

The Linux distribution is based on Debian plus the marian computer nonsense.ws environment we provide additional semi-autonomous assistance in a tiling window user interface, there is heavy focus on integration with Erlang/OTP and the official native implementation of ZeroMQ enabling zillions of Lua processes to communicate with Torch, PyTorch, OpenResty, LÃ–VE, Tornado, Turbo and all the Erlang, Python and Lua friendly multi-paradigm programming language ecosystems.

We make a stable workspace for distributed computing that anyone can use today, at nonsense.ws we put these things together into one unified environment with native support for Luerl an implementation of Lua written in pure Erlang.
