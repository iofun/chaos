SHA/OS HPC Linux

Sat 11 March 2017

SHA/OS HPC Linux is a nonsense.ws sponsored project. Our primary user base is within the distributed open-source high-performance computing community. However, our users come from a wide variety of industries.

Our Mission: Driven by nonsense.ws we focus on experimental research facilities, shaos.ws provides a professional workspace environment for creative research and distributed high-performance computing.

Our Goal: Provide a stable, scalable, and extensible distributed workspace environment for kids of all ages.

SHA/OS HPC Linux distribution, is based on Debian network install the UI and voice assistant is build around the lightweight i3 tiling window manager, the BEAM Virtual Machine and it's only possible thanks to the official high quality support for ZeroMQ within Erlang/OTP that enables millions of luerl Lua processes to communicate with torch, scikit-learn, pytorch and all the tools that valued eigens.

For information about how to get help or get involved see the Community Wiki.

We make a stable workspace for distributed high-performance computing that anyone can use today, at nonsense.ws we put these things together into one unified environment.
